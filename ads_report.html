▶ 사용 가이드
· 블럭 클릭 → 텍스트 수정  |  ⠿ 드래그 — 순서 변경
· B — 볼드  |  ← → — 들여쓰기  |  ✕ — 삭제
· 뱃지 클릭 — 타입 변경 (텍스트 → 자동:BK MO → 자동:BK PC → 자동:GFA → 자동:ADV)
· 자동 타입 — 파일 업로드 데이터로 자동 치환. 직접 수정 불가
· 빈 블럭 추가 — 공란(줄바꿈) 처리
· 저장하기 — Firebase 저장 (다음에도 유지)
· 전체 적용하기 — 저장 + 자동 데이터 치환 + 미리보기 반영
· 이번만 적용하기 — 저장 없이 자동 치환 + 미리보기 반영
· 복사하기 — 치환된 본문 클립보드 복사
